3. System requirements:

You need to have Java 5 or greater and Eclipse 3.3 installed to run the Embedded Editor. The Embedded Editor has been tested by us mainly on Windows XP, Linux and Mac OS X, but as a pure Java application may work on other platforms supporting Java 5 or greater.

These release notes are an HTML file, the other documents are PDFs which can be read using Adobe Reader (http://www.adobe.com/).
